# AI Risk Model Reviewer

## Overview

Reviews ML risk models for financial services: fairness, compliance (ECOA, FCRA, GDPR, SR 11-7), audit trails. Streamlit UI.

## Setup

1. cd AI_Risk_Model_Reviewer
2. pip install -r requirements.txt
3. Copy .env.example to .env and set OPENAI_API_KEY

## Run Commands

streamlit run app.py
